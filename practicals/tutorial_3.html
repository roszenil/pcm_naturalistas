Plot the tree with tip points colored by trait

## First we create a dataset with our matches 
## In the first column we are going to re-add the names of the species (taxa) so we know how to check for errors in the future

trait.dataset= as.data.frame(cbind(species=matched$phy$tip.label,matched$dat))

# Okay now let's do a fancy plot!

p <- ggtree(matched$phy, layout="circular") %<+% trait.dataset +  # %<+% joins data to the tree
  geom_tippoint(aes(color = strict), size = 1) +
  theme_tree2() +
  ggtitle("Sexual Systems (strict)")

This is a problem because it is assuming 0 and 1 are continuous values they arent

Transforming the data to plot it correctly

In order to plot this correctly with the colors that we want we need to convert these data to discrete categories. We do this with this option

discrete.strict= as.factor(matched$dat$strict)

#and we add our new column to the dataset. Notice that I never delete data or replace one column for another

trait.dataset= cbind(trait.dataset, discrete.strict)

Now we can plot using colors that we like

p <- ggtree(matched$phy, layout="circular") %<+% trait.dataset +  # This weird function %<+% joins data to the tree
  geom_tippoint(aes(color = discrete.strict), size = 1) +
  theme_tree2() +
  ggtitle("Sexual Systems (strict)")+
  scale_color_manual(values = c("0" = "#127475", 
                                "1" = "#F2542D"))
print(p)
